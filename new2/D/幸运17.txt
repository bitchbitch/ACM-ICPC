                     幸运17
Problem description
   xxs今天非常开心，因为他学会了如何快速判断一个数是否能被3整除。放学后，月月和xxs一起回家，xxs对月月说今天老师教的方法
真好，月月一脸鄙视的看着他, 说了一句不就是把各个位相加之和看是否能被3整除么，我都已经学会怎么快速判断能否被17整除了。xxs
想了很久都没想出来怎么快速判断能否被7整除。一脸郁闷的回到家后，波杰哥问xxs今天怎么不开心。在知道xxs不开心的原因后，波杰哥
告诉了xxs一个方法，设a是一个正整数且a>=0, b= a/10, c= a%10, 当(a-5*b)是17的倍数时，a是17的倍数。例如201不是17的倍数，因为
20-5=15不是17的倍数，68是17的倍数，因为6-40=-34是17的倍数。现在给你一个正整数，判断它是否是17的倍数。

Input
输入有多组数据，每组占一行，由一个正整数N（1<=N<=10^100）组成。


Output
对于每组测试数据，输出“YES”或者"NO"，"YES"表示N是17的倍数，“NO"表示不是。


Sample Input
17
201

Sample Output
YES
NO
