  有一段由小写字母'a'-'z'组成的字符串,它每秒会更新一次直到更新结束,他每
  一次更新的步骤如下:

   1,自己复制出一个和自己一样的新的字符串.
   2,新字符串反转以后再接到原字符串的尾部组成一个长度为原来两倍的新字符串.
   (例如: abc 更新一次以后变为 abccba)

  但是因为某些原因,字符串的复制会产生错误,例如 'a' 复制 成'b','z' 复制成'x'
  (但复制出的新字符串长度一定等于原串.) 给你该字符串的更新前的长度和它更新结束后的
  字符串,请你求出它的更新持续了多少秒和更新过程中产生的错误次数.
   
  输入:
   输入数据有多行组成,第一行是一个整数T(1 <= T <= 1000)表示数据的个数.
   接下来有T行,每一行由一个整数n(1 <= n <= 30)表示原字符串的长度,一个
   字符串str(长度为 1 - 40000)表示更新结束后的字符串组成.
  输出:
   输出数据由T行组成,每一行两个整数,第一个数表示该数据原字符串更新了
   多少秒,第二个数表示更新过程中产生了几次错误,两个数用空格隔开.
  
  样例输入:
   2
   3  abccba
   2  abbcabba

  样例输出:
   1 0
   2 2
