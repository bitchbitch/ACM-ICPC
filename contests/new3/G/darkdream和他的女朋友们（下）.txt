Problem Description

大家都知道darkdream是全宇宙最帅的男人，当然也有很多的女朋友啦。他给他的每个女朋友估计了一个魅力值Ai，有一天，他突然很想知道，
在他如此多的女朋友里面，魅力值的差别究竟有多大，于是，他把他的n个女朋友们召集到一起，从左到右排成一队，编号从左到右为1到n。
从左到右的魅力值依次是A1,A2,A3.....An，darkdream求的魅力值差可能有点不一样.他求的是Ai - Aj的最大值，且i < j。现在请你来写
一个程序来帮这个全宇宙最帅的男人来完成这个工作吧。
 

Input

输入首先是一个整数n，表示darkdream女朋友的数量(2<=n <= 100000)，接下来一行有n个整数Ai，0<=Ai<=1000000000，请处理到文件结束。

Output

输出一行Ai - Aj的最大值，i < j.
 

Sample Input
4
1 2 3 4
4
4 3 1 2
2
1 2

Sample Output

-1
3
-1
