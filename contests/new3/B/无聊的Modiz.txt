		无聊的Modiz
实验室ACM组最无聊的就是Modiz了，每天想着怎么玩得开心。今天他想到一个丢石子的游戏，首先他在地上画了三个点组成一个三角形，然后他站在很远的地方往三角形内丢石子。因为Modiz是个近视眼，所以不知道石子是否被丢在三角形内（擦边也算），所以想请实验室的各位大神帮帮他！

注：已知三边a,b,c 求三角形面积公式为 海伦公式
设 p = (a+b+c)/3
面积 = sqrt(p*(p-a)*(p-b)*(p-c))

输入样例：
第一行有6个数字，分别表示三角形A,B,C三点的坐标，
第二行表示表示Modiz丢石子的次数N。
接下来N行表示石子S的坐标。

输出样例：
如果石子在三角形内则输出Yes,否则输出No。

输入样例：
0 0 0 5 7 5
4
0 0
1 1
-1 -1
7 8

输出样例：
Yes
Yes
No
No
