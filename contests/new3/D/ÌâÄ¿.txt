				挑战恶龙
Problem Description
	《霍比特人》电影三部曲发生在《魔戒》故事60年前的中土大陆；讲述霍比特人比尔博・巴金斯被迫卷入向巨龙史矛戈宣战，
拯救孤山矮人王国的冒险之旅。在巫师甘道夫的怂恿下，内心喜爱冒险的他加入了由传奇战士索林・橡木盾带领的矮人兵团。他们的
旅程要深入荒野，横越哥布林及半兽人、座狼与魔法师潜伏之地。在历经重重困难之后，矮人兵团终于来到了孤山，那么问题来了，
怎样才能在牺牲最少的情况下击杀巨龙史矛戈呢？（后面纯属虚构）聪明的darkdream想出了一个好办法，从兵团中选出战斗力高的
那些人去击杀巨龙。现在给你兵团的人的战斗力，请输出按战斗力排名的排行榜。


Input
输入包括多个测试实例。
对于每个实例，首先是一个正整数n(n <= 3000)，接下来的n行每行有一串由小写字母构成的字符串Si(长度<= 1000)和一个正整数Ai(0<= Ai<= 10000)，Si表示第i个勇士的名字,Ai表示第i
个勇士的战斗力。输入保证任意两个勇士的名字，战斗力都不相同。

Output
第一行输出“Case #x:”，x表示第x组测试数据，接下来的n行表示战力排行榜。


Sample Input
3
darkdream 3131
wunaideweixiao 6119
wobulude 3932

Sample Output
Case #1:
wunaideweixiao
wobulude
darkdream
